# Product

## Register

product

## Users

GamePanel Lite is for players and small server owners who self-host Terraria servers and want a lightweight panel for day-to-day server operations.

## Product Purpose

GamePanel Lite manages local Docker-backed game server instances, starting with Terraria and tModLoader. Success means users can create, run, inspect, back up, restore, and join servers without handling container details directly.

## Brand Personality

Modern, lightweight, gamer-friendly. The interface should feel focused and capable, closer to a polished game launcher than an enterprise admin template.

## Anti-references

Avoid glassmorphism, heavy neon, cyberpunk decoration, childish pixel art, AI-style gradients, and dense enterprise settings pages.

## Design Principles

- Keep the common path visible and push rare configuration behind progressive controls.
- Use green only for primary actions and healthy status; reserve purple for modded Terraria and gold for warnings.
- Let operational feedback be immediate, short, and close to the action that caused it.
- Keep Docker and runtime details available without making them the first thing users must parse.

## Accessibility & Inclusion

Target clear contrast on dark surfaces, keyboard-accessible controls, readable status text, and reduced visual noise for repeated operational use.
